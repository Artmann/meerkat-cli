# Meerkat CLI

Meerkat is a CLI tool that suggests internal links that you can add to your website.

## Usage

```bash
npx meerkat-cli <sitemap-url> <url>

```

## Development

To install dependencies:

```bash
bun install

```

To run:

```bash
bun run src/index.ts

```

To run tests:

```bash
bun run test

```

To build

```bash
bun run build

```
